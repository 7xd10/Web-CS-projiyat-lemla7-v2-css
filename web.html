Module d'Études : Sécurité Web

Description

Ce module couvre les principes et les pratiques de la sécurité des applications web, y compris les vulnérabilités courantes, les techniques de protection et les bonnes pratiques pour assurer un web sécurisé.

Objectifs du Module

  • Comprendre les vulnérabilités des applications web (XSS, injection SQL, CSRF, etc.)
  • Appliquer les mécanismes d’authentification et d’autorisation sécurisés
  • Mettre en place des bonnes pratiques de sécurisation des données
  • Effectuer des audits de sécurité sur des applications web

Contenu du Cours

  • Introduction à la sécurité web
  • Les attaques web courantes et leurs mécanismes
  • Contrôles d'accès, gestion des sessions et authentification
  • Chiffrement des données et communications sécurisées (HTTPS, TLS)
  • Tests d’intrusion et audits de sécurité
  • Normes et bonnes pratiques (OWASP Top 10)

Modalités d'Évaluation

L’évaluation se base sur :

  • Travaux pratiques : 10 %
  • Projet de groupe : 10%
  • controle continu : 30%
  • Examen final : 50%

Durée et Organisation

Le module dure 10 semaines avec 3 heures par semaine (1h30 de cours + 1h30 de TP).

Année universitaire : 2025-2026

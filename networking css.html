Module d'Études : Réseaux Informatiques

Description

Ce module présente les concepts fondamentaux des réseaux informatiques, y compris les protocoles, les architectures réseau, la sécurité, et les services essentiels utilisés dans les infrastructures modernes.

Objectifs du Module

  • Comprendre les modèles OSI et TCP/IP
  • Identifier les principaux protocoles réseau
  • Configurer des réseaux simples
  • Appliquer les bases de la sécurité réseau

Contenu du Cours

  • Introduction aux réseaux (LAN, WAN, Internet)
  • Modèles OSI et TCP/IP
  • Adressage IP et sous-réseautage
  • Protocoles courants : HTTP, FTP, DNS, DHCP
  • Matériel réseau : routeurs, commutateurs, points d'accès
  • Principes de base en cybersécurité

Modalités d'Évaluation

L’évaluation se base sur :

  • Travaux pratiques : 10 %
  • Projet de groupe : 10%
  • controle continu : 30%
  • Examen final : 50%

Durée et Organisation

Le module dure 12 semaines avec 4 heures par semaine (2h de cours + 2h de TP).

Année universitaire : 2025-2026
